public static void main(String[] args) : 시작 전 필수입력(main + tap)
System.out.println(); : 출력코드
|n : 줄바꿈 문자
문자 연결 : "" + ""
숫자 더하기 : 숫자 + 숫자
문자와 숫자는 + 연산 결과가 다르므로 구별해서 사용해야 한다.
"숫자" + "숫자" = 숫자 + 숫자 / 숫자 + 숫자 = 숫자

변수 : 데이터를 저장하는 공간(그릇)
변수선언 : 변수타입 변수명;
변수에 값 저장 : 변수가 왼쪽, 저장값이 오른쪽
ex) int a = 50;
int : 숫자열(정수)
double : 숫자열(실수)
string : 문자열
boolean : 논리값

변수 사용시 주의사항
변수명은 중복되지 않도록 지어야 한다.
변수명은 숫자로 시작하면 안된다.
변수는 재활용이 가능하다.

산술연산자
+, -, *, /, %
정수와 정수의 계산 결과는 정수, 실수로 계산하려면 실수로 나누기

비교연산자
>, <, >=, <=, ==, !=
true, false >> 논리값(boolean)으로 결과가 나온다.

논리연산자
&&(and) : 둘 다 true 일때만 true, 그 외에는 false.
||(or) : 둘 다 false 일때만 false, 그 외에는 true.

조건문
if(실행조건){실행문}
else if문을 사용하면 연산상의 흐름성과 조건식의 간결함을 얻을 수 있다.
else 같은 경우는 마지막에 조건을 생각하고 사용가능(조건문 덩어지 마지막에 와야 함)
